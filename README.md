# node-mkvtools

[![NPM Version][npm-badge]][npm-url] [![Build Status][travis-badge]][travis-url]

This is a simple wrapper for the `MKVToolNix` cross-platform tools.

## Installation

```sh
$ npm install mkvtools
```
